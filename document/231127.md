## 싱글 파일 컴포넌트
--- 
js, html, css를 하나의 파일에 캡슐화하여 사용할 수 있디.

- 컴포넌트 범위의 css를 사용할 수 있다.
- 

```js
<script>
export default {
  data() {
    return {
      count: 0
    }
  }
}
</script>

<template>
  <button @click="count++">숫자 세기: {{ count }}</button>
</template>

<style scoped>
button {
  font-weight: bold;
}
</style>
```
